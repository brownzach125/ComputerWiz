Basic Spells

  • MAGIC.castFireBall(direcion , speed , radius)
    Lob a fireball
    Direction is degree from x axis
  • MAGIC.getPOS()
    Returns an object { x: xcoord , y: ycoord}
    that is the position of the wizard.
  • MAGIC.getOpponentPOS()
    Returns an object { x: xcoord , y: ycoord}
    that is the position of the opposing wizard.
  • BASIC.moveToPOS(x , y )
    Move your wizard to x , y position
  • BASIC.getFireBallsPOS()
    Returns a list of objects { position : { x : x , y :y}}
    That are the positions of all fireballs on the map.
